# 差异评审

APPROVED

## 范围核对

- 所有新增运行时能力都由 `start_client` 自动创建和释放，没有用户侧辅助命令或独立进程。
- 未更改客户端与服务端 WebSocket 协议，也未修改服务端推理和模型配置。
- uv 仅用于维护者构建，普通用户继续使用 EXE/ZIP。

## 正确性核对

- 短按取消事件会在 180ms 显示延迟内回到 idle。
- 录音、识别、送达和异常共用任务 ID，旧任务事件不能覆盖新录音状态。
- 送达事件位于文字输出成功返回之后。
- UI 在线程内运行，初始化异常被捕获，不会中断识别主流程。
- 客户端退出会发送 STOP 并等待 UI 线程结束。

## 构建核对

- 完整包与纯客户端包均成功通过 PyInstaller。
- 两个 ZIP 均包含 `start_client.exe`、`config_client.py`、`_tkinter.pyd` 和状态岛源码。
- ZIP 构建失败会返回非零退出码。

## 已知限制

- 状态岛使用当前主屏幕尺寸定位到屏幕底部中央；不同 DPI 和多屏位置仍建议在目标机器上实测。
- 本次没有连接真实 ASR Server 做端到端语音输入，状态事件边界由单元测试、导入检查和构建验证覆盖。
