# VoxCaps 3.0.2

这是 3.0.1 的 Server Worker 启动修复版本。VoxCaps 基于 [HaujetZhao/CapsWriter-Offline](https://github.com/HaujetZhao/CapsWriter-Offline) 二次开发，不是原项目的官方发行版。

## 修复内容

- 修复 Qwen ASR 启动时无条件导入可选 `sherpa_onnx`，因无关 DLL 加载失败导致 Worker 退出的问题；
- ASR 引擎现在只加载自身实际需要的可选依赖；
- Worker 的 Python 启动异常会回传主进程，并写入 `logs/worker_crash_latest.log`；
- 增加 `faulthandler` 原生崩溃日志 `logs/worker_native_crash_latest.log`；
- Worker 启动失败后主进程不再继续启动 WebSocket 服务，避免出现假启动状态；
- 增加可选 sherpa 依赖和 Worker 崩溃诊断回归测试。

## 下载

- `VoxCaps-20260720.zip`：Windows 完整包，包含客户端、服务端以及 GGUF/Vulkan 运行库；
- `VoxCaps-Client-20260720.zip`：Windows 纯客户端包，适合连接局域网服务器。

模型文件继续从原项目的 [Models Release](https://github.com/HaujetZhao/CapsWriter-Offline/releases/tag/models) 下载，并解压到完整包的 `models` 对应目录。
