## 使用问题

### 为什么在某些软件中无法输入或输入异常？

**情况一：完全无法输入**

部分程序以管理员权限运行，阻碍了客户端对键盘的监听。若想在这些程序上正常使用，可以用管理员权限运行客户端。

**情况二：能输入但标点符号后面的字不正常**

有两个原因：

1. **GUI 框架 IME 兼容性问题**：一些 GUI 框架的 IME 部分与系统 Unicode 输入字符不兼容。可以在 [config_client.py](../config_client.py) 的 `paste_apps` 列表中配置，强制在这些程序上使用粘贴输出。
2. **中文输入法干扰**：一些中文输入法（例如微信输入法）会干扰系统的 Unicode 字符输出。切换到英文键盘就能正常输出。解决方法：要么换别的输入法，要么在 [config_client.py](../config_client.py) 中改成粘贴输出。

### 按下按键没说话，松开后却输出了一些文字？

这是模型本身能力的限制。就算环境再安静，麦克风也会采集到微弱的底噪，强制让模型输出转录文字，模型就会产生幻觉（输出不存在的词句）。

如果对音量过低的音频直接不转录，又会导致悄悄话等低音量场景无法使用。所以只能使用时注意：真的需要说话打字时才按快捷键，不说话就别长按。

### 为什么没法打开麦克风？

1. 设置 > 隐私和安全性 > 麦克风，将「允许桌面应用访问麦克风」打开
2. 状态栏右下角音量图标 > 右键菜单 > 声音 > 麦克风的属性，关闭「允许应用程序独占控制该设备」
3. 状态栏右下角音量图标 > 右键菜单 > 声音 > 麦克风的属性，关闭「增强效果」
4. 如果安全软件或系统策略拦截了桌面应用访问麦克风，请放行 `start_client.exe`

VoxCaps 不会因为一次失败而退出；修改权限后，下一次长按 CapsLock 会再次尝试打开麦克风。

### 为什么 Windows 的麦克风指示会短暂出现？

客户端启动约 1 秒后会执行一次“打开、验证、立即关闭”的预热。平时按下 CapsLock 时也会立即异步打开麦克风并暂存开头音频；如果在 250 毫秒内松开，候选音频会被丢弃、麦克风随即关闭，并正常切换大小写。因此启动后或短按时，Windows 麦克风指示可能短暂闪现，这是正常现象。

预热结束、短按释放、长按录音结束或客户端退出后都会关闭设备。空闲期间不会持续占用麦克风。

### asset 文件夹里的录音文件会不会越积越多？

连续 24 小时不间断录音，体积也只有 2GB 左右。实测一个月下来录音文件不到 200MB，一年也不会超过 3GB，还没有微信聊天记录增长快，几年清理一次也不成问题。

所以自动删除录音的机制不会加。

## 不会做的功能

### 能实时显示预览/流式识别吗？这样就能看着文字确认自己说对了

这个需求想法很好，但实践上不会很好，不做。原因有二：

1. **准确率差**：当下的流式识别模型，不光有一到两秒的延迟，而且准确率会比非流式模型差很多。因为流式识别模型只能看到时间轴左侧的信息，看不到右侧的信息，上下文缺失了右侧的信息，准确率天然受限。
2. **输入效率降低**：流式识别输入需要先确定光标位置，再开始语音输入。而非流式识别可以先按下快捷键开始说话，鼠标再慢悠悠地找到需要输入的位置，点击切换光标，说完话松手上屏，效率更高。

CapsWriter 只专注于把输入做准、做快。流式识别在这里被判处永久死刑了，永远不会做。

### 屏幕上的动态状态岛分别表示什么？

- 琥珀波形：长按已经确认，正在等待麦克风就绪；
- 青蓝波形：麦克风已就绪，正在录音；
- 紫色光点：正在识别或处理结果。

短按 CapsLock 不显示状态岛；成功、取消或失败后直接隐藏，不显示绿色完成动画或红色错误叹号。

### 为什么不做成傻瓜式软件，打开后自动下载缺失的模型？

当下软件和模型是分开打包的：软件二进制约 100MB，每个模型从 500MB 到 1.5GB 不等。正常情况只有软件会更新，模型不需要更新，分开打包有利于减小更新体积。

分发渠道有两个：**GitHub Releases**和 **百度网盘**。

我也希望软件能检测到缺失模型后自动下载，但实际不可行：

- **GitHub**：大部分中国大陆用户访问 GitHub 不太顺畅，内置从 GitHub Releases 下载几百 MB 甚至 1GB+ 的模型文件，速度堪忧。
- **百度网盘**：国内下载比较稳，但百度网盘不提供第三方 API 接口供程序调用下载。

所以只能劳烦用户，要么用加速器到 GitHub Releases 下载模型，要么到百度网盘客户端中打开分享链接下载模型。

### 按快捷键录音时，能自动静音电脑扬声器，松开再恢复吗？这样能防止扬声器干扰麦克风

这个我不会做。建议使用桌面 USB 麦克风，50 块左右就能买到，麦克风离嘴近一些，比其他各种手段提升准确率都更有效得多。

再精妙的软件后期处理，也不如物理上的前期处理更有效。
